# PlanetCamera
可以在一般平面、球上、甜甜上使用的第3人稱角色控制

# 操作

鍵盤  
方向：WASD  
加速：J  
跳：Space  

滑鼠  
加速：左鍵  
跳：右鍵  

PS4手把  
跳：X  
加速、R2  

靠牆可以再跳一次  
不過目前沒特別做演出的動畫
